# rbShops
• Script de superettes simple optimisé, et sécurisé  
• Framework: ESX  
• Lib utilisée: RageUI V2 (Modifié par mes soins)  

# Installation
• Mettre rbShops dans votre dossier resources  
• Start la resource dans server.cfg
